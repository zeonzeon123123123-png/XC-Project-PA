# 🤖 AI 智能表单助手 - 使用指南

## 📋 功能介绍

AI 智能表单助手是一个集成在研发项目文档系统中的智能辅助工具，提供实时填写建议和问答功能。

### ✨ 核心功能

1. **实时表单建议** 💡
   - 自动监测用户正在编辑的字段
   - 基于历史项目数据提供智能建议
   - 支持一键应用建议到表单

2. **交互式问答** 💬
   - 用户可随时提问
   - AI 结合历史项目资料回答
   - 支持联网搜索（预留接口）

3. **历史数据参考** 📚
   - 内置 5 个真实历史项目数据
   - 自动检索相关项目作为参考
   - 提供统计数据（平均预算/周期等）

## 📖 使用教程

### 1. 实时表单建议

**步骤：**
1. 打开研发立项书表单
2. 点击任意表单字段（如"预算金额"）
3. AI 助手会自动弹出建议面板
4. 查看建议并选择：
   - ✅ **应用建议**：一键填充到表单
   - 📝 **修改建议**：手动调整后使用
   - ❌ **忽略**：继续自己填写

**示例：**
```
当你点击"预算金额"字段时：

💰 预算建议
根据 5 个历史项目数据：
• 平均预算：850 万元
• 范围：75 万 - 2400 万元

[应用平均值] [查看示例]
```

---

### 2. 问答对话

**步骤：**
1. 在 AI 助手面板底部找到输入框
2. 输入你的问题
3. 按 Enter 或点击发送按钮
4. AI 会结合历史数据回答

**示例问题：**
```
❓ "研发立项书中的技术可行性应该怎么写？"

🤖 AI 回答：
📚 根据历史项目数据：

• IPC 摄像头自动化生产线 (SR1202500015)
  1. 团队具备 AI 和嵌入式开发技术积累
  2. 采用成熟芯片平台，风险可控
  3. 已有原型验证，技术路径清晰

• 基于 AI 高性能 GPU 算力集群的管理开发平台 (SR1202500001)
  1. 团队具备 AI、分布式系统、高性能计算等领域技术积累
  2. 采用分阶段、模块化开发模式，风险可控
```

---

### 3. 快捷操作

|------|------|------|

```json
{
  "projects": [
    {
      "id": "SR1202500015",
      "name": "项目名称",
      "type": "智能硬件",
      "budget": 750000,
      "duration": 12,
      "techGoal": "技术目标",
      "techFeasibility": "技术可行性",
      "necessity": "项目必要性"
```

```javascript
aiAssistant = new AIFormAssistant({
```

### 自定义扩展

**添加新的历史项目：**

编辑 `project-history.json`：
```json
{
  "projects": [
    {
      "id": "SR1202500XXX",
      "name": "你的项目名称",
      "type": "项目类型",
      "budget": 预算金额，
      "duration": 周期，
      "techGoal": "技术目标",
      "techFeasibility": "可行性分析",
      "necessity": "必要性分析"
    }
  ]
}
```

**添加新的建议类型：**

编辑 `js/ai-form-assistant.js`，在 `generateSuggestion` 方法中添加：
```javascript
case 'yourFieldType':
    return this.generateYourSuggestion(fieldValue);
```

## ❓ 常见问题

### Q: AI 助手没有自动弹出？
A: 确保你已经点击了表单字段，并且页面已完全加载。

### Q: 如何重新打开关闭的 AI 助手？
A: 刷新页面即可重新打开。

### Q: 历史数据从哪里来？
A: 来自过往的真实研发项目文档，已脱敏处理。

### Q: 可以禁用 AI 助手吗？
A: 可以点击面板右上角的 × 关闭，或刷新页面。
