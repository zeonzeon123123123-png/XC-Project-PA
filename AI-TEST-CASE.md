# 🧪 AI 智能表单助手 - 测试案例

## 📋 测试目标

验证 AI 智能表单助手的以下功能：
1. ✅ 实时表单建议（字段聚焦时自动弹出）
2. ✅ 联动建议（结合已填写的其他字段）
3. ✅ 问答对话功能
4. ✅ 历史数据检索
5. ✅ 建议应用功能

## 🐛 已知问题排查

### 问题 1：AI 面板不显示

**可能原因：**
- 页面未完全加载
- 浏览器控制台有 JavaScript 错误

**解决方法：**
1. 刷新页面
2. 按 F12 打开浏览器控制台
3. 查看是否有红色错误
4. 确认 `ai-form-assistant.js` 已加载

---

### 问题 2：建议不显示内容

**可能原因：**
- 历史数据加载失败
- 字段类型识别失败

**解决方法：**
1. 打开浏览器控制台
2. 查看日志输出：
   ```
   ✅ AI Form Assistant 已初始化
   ✅ 历史数据已加载
   📝 字段聚焦：budget
   🔍 字段类型：budget
   ```
3. 确认历史数据成功加载

---

### 问题 3：联动建议不工作

**可能原因：**
- 表单字段 ID 不匹配
- 已填写数据未正确读取

**解决方法：**
1. 确认已填写其他字段（如项目名称、类型）
2. 查看控制台日志：
   ```
   📊 当前表单数据：{ projectName: "XXX", projectType: "XXX" }
   ```
3. 确认数据已正确读取
